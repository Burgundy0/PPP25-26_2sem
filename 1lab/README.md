# Терминальные шахматы

Классические шахматы для терминала с оптимизированным ядром на C.

## Требования

- GCC (для компиляции C-библиотеки)
- Python 3.6+

## Установка

```bash
# Скомпилировать библиотеку
gcc -shared -fPIC -O3 -o libboard.so Board.c
```

## Запуск игры

```bash
python3 chess.py
```

## Управление

### Формат ввода ходов

Вводите ход в формате **откуда-куда**:
- `e2e4` — пешка с e2 на e4
- `g1f3` — конь с g1 на f3
- `e1g1` — короткая рокировка белых
- `e7e8` — превращение пешки (автоматически в ферзя)

Также поддерживается формат с дефисом: `e2-e4`

### Команды

| Команда | Описание |
|---------|----------|
| `quit`, `exit`, `q` | Выход из игры |
| `Ctrl+C` | Прервать игру |

## Правила

Поддерживаются все стандартные правила шахмат:
- ✅ Ходы всех фигур
- ✅ Рокировка (королевская и ферзевая)
- ✅ Взятие на проходе (en passant)
- ✅ Превращение пешки (автоматически в ферзя)
- ✅ Шах, мат, пат
- ✅ Нельзя ходить под шах

## Оптимизация памяти

Ядро на C использует минимальную память:
- **1 байт** на клетку (64 байта на доску)
- **4 бита** на фигуру (3 бита тип + 1 бит цвета)
- **74 байта** полная структура позиции

```bash
# С отладочной информацией
gcc -shared -fPIC -g -o libboard.so Board.c

# С максимальной оптимизацией
gcc -shared -fPIC -O3 -o libboard.so Board.c
```
